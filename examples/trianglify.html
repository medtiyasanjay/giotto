Trianglify

The trianglify visualization uses the Trianglify javascript library.

Html:

<div data-options="gexamples.trianglify" data-giotto-trianglify>

Javascript code:

gexamples.trianglify = {
    height: 400,

    onInit: function (viz) {

        luxforms.redraw = function (e) {
            e.preventDefault();
            if (this.form.$valid)
                viz.redraw(this.trianglify);
        };
    }
};
